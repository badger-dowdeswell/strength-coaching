Installing the latest Node.js v24.0.2

These instructions are available at https://nodejs.org/en/download 

https://nodejs.org/en/download and switch to the Linux tab for the prebuilt Node.js

Run this in the terminal:

# Download and install nvm:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# Download and install Node.js:
nvm install 24

# Verify the Node.js version:
node -v # Should print "v24.0.2".

nvm current # Should print "v24.0.2".

# Verify npm version:
npm -v # Should print "11.3.0".
